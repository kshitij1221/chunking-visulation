# chunking-visulation
this is application is to show how chunking is working with text data.


While learning about vector databases, I came across one concept that appears everywhere in generative AI: chunking.
Chunking is one of the most important techniques in both generative AI and vector databases. At first, I was confused about how exactly it is used.
To understand it better, I built a chunking visualizer - an interactive tool where you can input your own text and immediately see how chunking actually works under the hood.
With this visualizer, you can experiment with different chunking strategies, observe how the text is split, and get an intuitive feel for how it affects retrieval and generation in GenAI systems.


how to visulasize it:
just run .html file and you can easily do the chunking and see the chunks.
